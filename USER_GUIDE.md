## Getting Started

### First Time Opening the App

When you first open ProbablyTasty, you'll see two panels:
- **Left side**: A list showing all your recipes organized by tags

The window is divided by a movable splitter - drag the divider left or right to adjust the size of each panel.

### The Main Screen Layout

**Top Menu Bar:**
- **File**: Create new recipes, import from files, export recipes
- **Edit**: Edit or delete the currently selected recipe
- **Tools**: Recipe scaling, unit conversion, shopping lists
- **Settings**: Change themes and preferences
- **Help**: View this guide and app information

**Left Panel:**
- Search box at the top

- Quick action buttons at the bottom (New Recipe, Import, etc.)

**Right Panel:**
- Recipe title and details
- Ingredients list
- Step-by-step instructions

## Adding Recipes

There are several ways to add recipes to ProbablyTasty:

### Method 1: Create a New Recipe Manually

1. Click the **"+ New Recipe"** button (bottom of left panel) OR go to **File ‚Üí New Recipe**
2. Fill in the recipe form:
   - **Title**: Give your recipe a name
   - **Description**: A short summary of the dish
   - **Servings**: How many people it serves
   - **Prep Time**: How long to prepare (in minutes)
   - **Cook Time**: How long to cook (in minutes)
   - **Instructions**: Step-by-step cooking directions
   - **Source URL**: (Optional) Where you found the recipe
3. Add ingredients one at a time:
   - Enter the quantity (like "2" or "1/2")
   - Enter the unit (like "cups" or "tablespoons")
   - Enter the ingredient name
   - (Optional) Add preparation notes like "diced" or "melted"
   - Click **"Add Ingredient"**
4. Add tags to organize your recipe (see [Using Tags](#using-tags))
5. Click **"Save"** when you're done

### Method 2: Import from a Website

This is the easiest way if you found a recipe online!

1. Copy the recipe's web address (URL) from your browser
2. In ProbablyTasty, click **"üîó Import URL"** OR go to **File ‚Üí Import from URL**
3. Paste the URL and click **"Import"**
4. The app will automatically extract:
   - Title and description
   - Ingredients with quantities
   - Cooking instructions
   - Cooking times
5. Review the imported recipe and make any adjustments
6. Click **"Save"**

**Note:** This feature requires an AI service to be configured (see [Settings](#settings))

### Method 3: Import from a Photo

Have a recipe in a cookbook or printed on paper? Take a photo!

1. Take a clear, well-lit photo of the recipe
2. In ProbablyTasty, click **"üì∑ Import Image"** OR go to **File ‚Üí Import from Image**
3. Select your photo file
4. The app will read the text and extract the recipe
5. Review and correct any details that weren't captured perfectly
6. Click **"Save"**

**Note:** This feature requires an AI service with vision capabilities (see [Settings](#settings))

### Method 4: Import from a File

If you have recipes saved in JSON format or HTML files exported from ProbablyTasty:

1. Go to **File ‚Üí Import from JSON** or **Import from HTML**
2. Select your file(s)
3. The recipes will be added to your collection automatically

## Finding Recipes

### Using the Search Box

The search box at the top of the left panel helps you find recipes quickly.

**Simple Search:**
- Just type a few words and press Enter
- Example: Type "chocolate" to find all recipes with chocolate

**Smart Search (with AI):**
If you have AI configured, you can search using natural language:
- "What can I make with chicken and rice?"
- "Show me quick weeknight dinners"
- "I want something spicy and vegetarian"

The app understands what you're looking for and finds matching recipes.

### Browsing by Tags

In the left panel, your recipes are organized under tag categories like:
- **Cuisine**: Italian, Mexican, Asian, etc.
- **Meal Type**: Breakfast, Lunch, Dinner, Dessert
- **Dietary**: Vegetarian, Gluten-Free, etc.

Click on a tag to see all recipes with that tag. Click on a recipe to view its details.

## Using Tags

Tags help you organize and find your recipes. Think of them like labels or categories.

### Adding Tags to a Recipe

When creating or editing a recipe:
1. In the **Tags** section, type a tag name
2. Press Enter or click the **"+"** button
3. Add as many tags as you want
4. Common tag ideas:
   - **Meal type**: breakfast, lunch, dinner, snack, dessert
   - **Cuisine**: Italian, Mexican, Chinese, Thai, Indian
   - **Main ingredient**: chicken, beef, pasta, vegetarian
   - **Occasion**: holiday, party, weeknight, quick
   - **Season**: summer, fall, winter, spring
   - **Dietary**: vegetarian, vegan, gluten-free, dairy-free

### Removing Tags

To remove a tag from a recipe:
1. Edit the recipe
2. Click the **"√ó"** next to the tag you want to remove

### Managing Tags

Tags are created automatically when you add them to recipes. The app groups recipes by these tags in the left panel, making it easy to browse your collection.

## Kitchen Tools

ProbablyTasty includes helpful tools for cooking:

### Recipe Scaling

Need to make more or less of a recipe? Scale it!

1. Open the recipe you want to scale
2. Click **"‚öñÔ∏è Scale"** button
3. Choose a scale factor:
   - **2x**: Double the recipe
   - **3x**: Triple the recipe
   - **1/2x**: Cut the recipe in half
   - Or enter a custom multiplier
4. The app calculates all new ingredient amounts
5. Choose to:
   - **Save as New Recipe**: Keep both the original and scaled version
   - **Update This Recipe**: Replace the original with the scaled version

### Unit Conversion

Convert between metric and imperial measurements:

1. Go to **Tools ‚Üí Unit Conversion**
2. Enter a value and unit (like "2 cups")
3. Select what you want to convert to
4. See the result instantly

Common conversions:
- Cups ‚Üî Milliliters
- Teaspoons/Tablespoons ‚Üî Milliliters
- Ounces ‚Üî Grams
- Fahrenheit ‚Üî Celsius (for oven temperatures)

### Shopping List

Planning to cook multiple recipes? Create a shopping list:

1. Go to **Tools ‚Üí Shopping List**
2. Select the recipes you want to cook
3. The app combines all ingredients
4. Similar ingredients are grouped together
   - Example: If Recipe A needs 1 cup flour and Recipe B needs 2 cups flour, the list shows "3 cups flour"
5. Print or export your shopping list

## Exporting & Sharing

### Export to HTML (Recommended for Sharing)

HTML exports create beautiful, self-contained recipe files that anyone can open in a web browser:

1. Open the recipe you want to share
2. Click **"üì§ Export"** and choose **"HTML"**
3. Save the file
4. Share it by email, cloud storage, or USB drive

**What's special about HTML exports:**
- Opens in any web browser (Chrome, Firefox, Safari, Edge)
- Works on phones, tablets, and computers
- No ProbablyTasty app needed to view it
- Interactive features:
  - Adjust serving sizes with automatic recalculation
  - Switch between metric and imperial units
  - Check off ingredients as you use them
  - Print-friendly layout
- Beautiful teal-themed design
- Can be imported back into ProbablyTasty later

### Export to Other Formats

**Markdown (.md):**
- Good for simple text editors
- Easy to read and edit
- Great for sharing in plain text

**JSON (.json):**
- Technical format for backup
- Can import multiple recipes at once
- Preserves all recipe data perfectly

**To export:**
1. Select the recipe(s) you want to export
2. Go to **File ‚Üí Export**
3. Choose your format
4. Select where to save the file

### Printing Recipes

To print a recipe on paper:

1. Open the recipe
2. Click the **"üñ®Ô∏è Print"** button
3. Choose your printer
4. Adjust print settings if needed
5. Click Print

The recipe will print in a clean, easy-to-read format.

## Settings

### Changing the Theme

ProbablyTasty comes with two beautiful themes:

1. Go to **Settings ‚Üí Preferences**
2. Click the **"Appearance"** tab
3. Choose **Light** or **Dark** theme
4. Click **"Apply"** to preview
5. Click **"OK"** to save

**Light Theme**: Clean white background, perfect for daytime use  
**Dark Theme**: Easy on the eyes, great for evening cooking

### Setting Up AI Features

To use smart search, photo import, and URL import:

1. Go to **Settings ‚Üí Preferences**
2. Click the **"LLM Settings"** tab
3. Choose your AI provider:
   - **Ollama** (Free, runs on your computer)
   - **OpenAI** (ChatGPT, requires API key)
   - **Anthropic** (Claude, requires API key)
   - **Google** (Gemini, requires API key)

**For Ollama (Recommended for Beginners):**
1. Download Ollama from ollama.ai
2. Install and run it
3. In ProbablyTasty settings, select "Ollama"
4. Choose a model like "llama2" or "mistral"
5. Click "Test Connection" to verify it works

**For Cloud AI Services:**
1. Sign up for an account with your chosen provider
2. Get an API key from their website
3. In ProbablyTasty settings, paste your API key
4. Click "Test Connection" to verify it works

**Note:** AI features are optional. You can use ProbablyTasty without them for manual recipe entry, browsing, and basic search.

## Troubleshooting

### Common Issues and Solutions

**Problem: The app won't start**
- Make sure Python is installed
- Check that all required packages are installed (run `pip install -r requirements.txt`)
- Try running from a terminal to see error messages

**Problem: Can't import from URL**
- Check your internet connection
- Make sure an AI provider is configured in Settings
- Some websites block automatic recipe extraction
- Try copying the recipe text manually instead

**Problem: Photo import isn't working**
- Ensure the photo is clear and well-lit
- Make sure an AI provider with vision support is configured (OpenAI, Anthropic, or Google)
- Ollama doesn't support photo import yet
- Try taking a better photo or typing the recipe manually

**Problem: Search isn't finding recipes**
- Check for typos in your search terms
- Try using different keywords
- Browse by tags instead
- For smart search, make sure AI is configured

**Problem: Tags aren't showing up correctly**
- Tags are created when you add them to recipes
- Make sure you pressed Enter after typing the tag name
- Tags appear in the left panel after you save the recipe

**Problem: Exported HTML file won't open**
- Right-click the file and choose "Open with" ‚Üí your web browser
- Make sure the file has a .html extension
- The file should work in any modern browser

**Problem: Can't see button text in the left panel**
- Try making the left panel wider by dragging the divider
- The default size should now be 50/50 and show all buttons
- You can maximize the window if needed

### Getting Help

If you're still having trouble:

1. Check the README.md file for installation instructions
2. Make sure you're using the latest version
3. Look for error messages in the terminal/console
4. Try restarting the app
5. Ask the person who set up the app for help
